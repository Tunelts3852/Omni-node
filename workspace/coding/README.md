# Omni-node Coding Project

이 프로젝트는 로깅 기능이 포함된 Python 모듈과 단위 테스트 환경을 포함하고 있습니다.

## 주요 구성 요소
- `hello.py`: 인사말 생성 및 로깅 기능을 제공하는 핵심 모듈
- `test_hello.py`: `unittest` 라이브러리를 사용한 기능 검증 스크립트

## 실행 방법

### 테스트 실행
워크스페이스 내에서 다음 명령어를 실행하여 기능을 검증할 수 있습니다:
```bash
python3 test_hello.py
```

### 로깅 확인
스크립트 실행 시 표준 출력(stderr)을 통해 로깅 정보를 확인할 수 있습니다.
