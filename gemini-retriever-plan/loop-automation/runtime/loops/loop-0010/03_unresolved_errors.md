# Loop 0010 - 미해결 오류

## 처리하지 못한 오류

- 현재 재현되는 치명 블로커 없음
- 비차단 백로그: guard webhook/log collector live URL 미설정 (로컬+텔레그램 운영 범위 밖, 사용자 요청 시에만 연동)
- 비차단 항목: guard 임계치 2차 튜닝 미반영(이번 루프에서 readiness `ready=true` 충족, 다음 루프에서 임계치 재산정 예정)
- 비차단 항목: `publishedAt` 누락 문서 기본 폐기 정책으로 일부 질의에서 count-lock 미충족 차단률이 남을 수 있음 (보강 정책 문서화/텔레그램 회귀 assert 강화/임계치 락 회귀 자동화/채널 범위 회귀 강화/샘플 누적 리셋 방지/샘플 30건 누적 완료, 2차 튜닝 예정)
