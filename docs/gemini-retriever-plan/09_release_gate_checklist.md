# P7 릴리스 게이트 체크리스트

업데이트 기준: 2026-03-13

## 배포 전 점검
- [ ] 문서/스크립트 경로가 `gemini-retriever-plan` 기준으로 일치한다.
- [ ] 금칙어 목록 0건 정책을 통과한다.
- [ ] 키 소스 정책(`keychain|secure_file_600`) 강제 검증을 통과한다.
- [ ] 환경변수 키 직접 주입 차단 검증을 통과한다.
- [ ] 대화/코딩/텔레그램(우회) 공통 파이프라인 검증을 통과한다.

## 기능 회귀 점검
- [ ] 요청 건수 N = 출력 건수 N
- [ ] 시간창 외 문서 출력 0건
- [ ] 인용 매핑 누락 없음
- [ ] fail-closed 실패 케이스 메시지 일관성 확보

## 운영 점검
- [ ] keySource 로그 기록 확인
- [ ] countLockSatisfied 로그 기록 확인
- [ ] freshness/credibility/coverage 로그 기록 확인
- [ ] dropReasons/retryCount 기록 확인

## 롤백 조건
- 키 소스 정책 위반 발생
- fail-closed 미동작
- count-lock 우회 출력 발생
- 채널 공통 파이프라인 회귀 실패
